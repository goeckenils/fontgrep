# .scratch/ — Local Issue Tracker
> Durable, file-based PRD + issue tracker for this project. Convention: one feature per directory, PRD at \`.scratch/<feature>/PRD.md\`, slices at \`.scratch/<feature>/issues/NN-<slug>.md\`. Triage state lives on each issue's YAML frontmatter (\`Status:\`). See \`docs/agents/issue-tracker.md\` and the Hermes skill \`local-issues-folder\`.
